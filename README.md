My blog
========
Another lightweight blog cms based on tornado.(Don't complete)

Prequisites
---
* CPython 2.7
* [Markdown](http://pythonhosted.org/Markdown)
* [Tornado](http://www.tornadoweb.org/)
* [Sqlite](http://www.sqlite.org/)

Deployment
---
	chmod +x entry.py
	sudo ./entry.py

Visit http://localhost/
